<div style="background-color: #fff5f7; border-radius: 16px; padding: 24px; margin-bottom: 30px; box-shadow: 0 10px 15px -3px rgba(0,0,0,0.05); border: 1px solid #c2255c1a; color: #212529;"><div style="border-left: 5px solid #c2255c; padding-left: 15px; margin-bottom: 15px;"><h3 style="margin: 0; color: #c2255c; font-size: 1.4em;">1. （お時間を）いただく <span style="font-size: 0.75em; font-weight: 800; color: #c2255c; border: 1.5px solid #c2255c; padding: 1px 8px; border-radius: 6px; margin-left: 10px;">謙譲語</span></h3></div><div style="display: flex; align-items: center; gap: 12px; margin: 18px 0;"><div style="background: #c2255c; color: white; padding: 6px 14px; border-radius: 8px; font-weight: 800;">（お時間を）いただく</div><div style="color: #c2255c; font-weight: bold;">➜</div><div style="color: #868e96; font-size: 0.8em; font-weight: 600;">PLAIN FORM</div><div style="color: #c2255c; font-weight: bold;">➜</div><div style="border-bottom: 3px solid #c2255c; color: #212529; padding: 2px 14px; font-weight: 800;">もらう</div></div><div style="margin-bottom: 10px;">🏢 <b>使用场景</b>：对上司、客户、前辈表示感谢，尤其是对方为了自己专门腾出时间进行交谈、面试或指导时。</div><div style="margin-bottom: 10px;">📘 <b>核心讲解</b>：“いただく”是“もらう”的谦让语。在此处表示对方赋予了自己宝贵的时间。通过降低自己的行为来表示对对方赋予行为的敬意。在职场中，“お時間をいただく”是请求或感谢对方拨冗接见时的标准用语。</div><div style="margin-bottom: 20px;">💡 <b>记忆技巧</b>：动词常用“いただく”，构成“（物品/时间/机会）をいただく”的固定搭配。</div><div style="background-color: rgba(255,255,255,0.7); border-radius: 12px; padding: 15px; border: 1px solid #c2255c22;"><div style="font-weight: 700; margin-bottom: 6px; font-size: 0.85em; color: #c2255c; opacity: 0.7;">原文例句</div><div style="font-size: 1.05em; color: #212529; line-height: 1.6;">本当にお忙しい中お時間いただき、ありがとうございました。</div><div style="font-size: 0.95em; color: #868e96; font-style: italic; margin-top: 4px;">非常感谢您在百忙之中抽空（给我时间）。</div></div></div>
