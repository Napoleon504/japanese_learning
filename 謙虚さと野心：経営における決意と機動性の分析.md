<div style="background-color: #fff5f7; border-radius: 16px; padding: 24px; margin-bottom: 30px; box-shadow: 0 10px 15px -3px rgba(0,0,0,0.05); border: 1px solid #c2255c1a; color: #212529;"><div style="border-left: 5px solid #c2255c; padding-left: 15px; margin-bottom: 15px;"><h3 style="margin: 0; color: #c2255c; font-size: 1.4em;">1. ご評価いただける <span style="font-size: 0.75em; font-weight: 800; color: #c2255c; border: 1.5px solid #c2255c; padding: 1px 8px; border-radius: 6px; margin-left: 10px;">謙譲語</span></h3></div><div style="display: flex; align-items: center; gap: 12px; margin: 18px 0;"><div style="background: #c2255c; color: white; padding: 6px 14px; border-radius: 8px; font-weight: 800;">ご評価いただける</div><div style="color: #c2255c; font-weight: bold;">➜</div><div style="color: #868e96; font-size: 0.8em; font-weight: 600;">PLAIN FORM</div><div style="color: #c2255c; font-weight: bold;">➜</div><div style="border-bottom: 3px solid #c2255c; color: #212529; padding: 2px 14px; font-weight: 800;">評価してもらう</div></div><div style="margin-bottom: 10px;">🏢 <b>使用场景</b>：在向外部（客户或社会）描述自己公司获得的成就或声誉时使用。</div><div style="margin-bottom: 10px;">📘 <b>核心讲解</b>：“ご…いただく”是“もらう”的自谦表现，表示蒙受对方的恩惠。这里表达了“承蒙客人的认可”这种卑下的心态，强调成功的功劳在于客户的认可。</div><div style="margin-bottom: 20px;">💡 <b>记忆技巧</b>：ご + 名词 + いただける（可能态自谦），表示“有幸能得到……”</div><div style="background-color: rgba(255,255,255,0.7); border-radius: 12px; padding: 15px; border: 1px solid #c2255c22;"><div style="font-weight: 700; margin-bottom: 6px; font-size: 0.85em; color: #c2255c; opacity: 0.7;">原文例句</div><div style="font-size: 1.05em; color: #212529; line-height: 1.6;">やっぱり自分たちは何を持ってお客様にご評価いただけているのかということを忘れない。</div><div style="font-size: 0.95em; color: #868e96; font-style: italic; margin-top: 4px;">我们不能忘记，自己究竟是凭什么得到了客人的评价（认可）。</div></div></div>

<div style="background-color: #fff5f7; border-radius: 16px; padding: 24px; margin-bottom: 30px; box-shadow: 0 10px 15px -3px rgba(0,0,0,0.05); border: 1px solid #c2255c1a; color: #212529;"><div style="border-left: 5px solid #c2255c; padding-left: 15px; margin-bottom: 15px;"><h3 style="margin: 0; color: #c2255c; font-size: 1.4em;">2. 足を運んでくれている <span style="font-size: 0.75em; font-weight: 800; color: #c2255c; border: 1.5px solid #c2255c; padding: 1px 8px; border-radius: 6px; margin-left: 10px;">尊敬語</span></h3></div><div style="display: flex; align-items: center; gap: 12px; margin: 18px 0;"><div style="background: #c2255c; color: white; padding: 6px 14px; border-radius: 8px; font-weight: 800;">足を運んでくれている</div><div style="color: #c2255c; font-weight: bold;">➜</div><div style="color: #868e96; font-size: 0.8em; font-weight: 600;">PLAIN FORM</div><div style="color: #c2255c; font-weight: bold;">➜</div><div style="border-bottom: 3px solid #c2255c; color: #212529; padding: 2px 14px; font-weight: 800;">来る</div></div><div style="margin-bottom: 10px;">🏢 <b>使用场景</b>：在服务业或商务中，提及客户到访时必用的地道表达。</div><div style="margin-bottom: 10px;">📘 <b>核心讲解</b>：“足を運ぶ”本身是特意前往的意思，“～てくれる”是对方为我做某事的敬语表达。合起来表示对顾客光临行为的敬意和感激。</div><div style="margin-bottom: 20px;">💡 <b>记忆技巧</b>：形容客人来访不说“来る”，而说“足を運ぶ”，再加上“くださる/くれる”表示感激。</div><div style="background-color: rgba(255,255,255,0.7); border-radius: 12px; padding: 15px; border: 1px solid #c2255c22;"><div style="font-weight: 700; margin-bottom: 6px; font-size: 0.85em; color: #c2255c; opacity: 0.7;">原文例句</div><div style="font-size: 1.05em; color: #212529; line-height: 1.6;">なぜ今日このお客さんは我が店に足を運んでくれているのかっていうところをね、やっぱりしっかり理解していく必要性がある。</div><div style="font-size: 0.95em; color: #868e96; font-style: italic; margin-top: 4px;">我们有必要深刻理解，为什么这位客人今天会特意光临我们店。</div></div></div>
